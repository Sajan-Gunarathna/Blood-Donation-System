# Blood-Donation-System

###Requirements

java 21 - OpenJDK21 is recommended
Apache maven - version 3.11.0 is recommended
###Set up the development server

Clone the git repository - https://github.com/Sajan-Gunarathna/Blood-Donation-System.git
Open the project in your favourite IDE. (I used IntelliJ IDEA 2025.2 )
Goto pom.xml and add missing packages.
Goto Preferences -> Plugins, Add lombok plugin.

###database Connection(we use MySQL Workbench 8.0 CE)
goto application.properties change your username and password and create database lifeline_db2



local - http://localhost:8080/
